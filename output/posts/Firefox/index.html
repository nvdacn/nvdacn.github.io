推荐一些方便日常使用的 Firefox 插件

酱油

2017-01-20 10:04

资源

NVDA 对于 Firefox 的支持非常好，在 Firefox 上的响应速度也很快，现在已经有很多 NVDA 用户开始使用 Firefox。下面就推荐一些方便我们日常使用 Firefox 的扩展。

Adblock Plus

被广告弄得搓火吗？被跟随鼠标的标语弄得烦心吗？被横幅弄得感到厌恶吗？立即安装 Adblock Plus 来重新取得 Internet 的控制权并且改变您浏览网页的方式。Adblock Plus是 Firefox 上的广告拦截插件，用户可下载或灵活的自定义拦截规则。 Firefox 用户点击进入 Adblock Plus下载页

Navigational Sounds

习惯了 Internet Explorer 打开网页时辅助功能的声音提示了吗？觉得 Firefox 没有声音事件遗憾吗？安装 Navigational Sounds 就能满足你的愿望。 Firefox 用户点击进入 Navigational Sounds 下载页

NetVideoHunter

这款扩展是一个辅助下载工具，它能帮助你从Youtube、Facebook、Metacafe、Dailymotion、Break、VKontakte、DivShare以及其他更多网站上轻松下载视频和音乐。 Firefox 用户点击进入 NetVideoHunter 下载页

xThunder

xThunder 可以管理常用下载器（Windows，Linux，Mac），并支持迅雷、快车、QQ旋风、电驴、磁力、飞速盘等专用链接解析。 Firefox 用户点击进入 xThunder 下载页

网页翻译

这是中国火狐社区提供的一款调用有道翻译进行网页英汉互译的插件。 点击转到中国火狐社区网页翻译插件介绍下载页

User Agent Switcher

想用电脑访问手机网站，想在电脑上玩手机页游，可是浏览器总是无法打开手机网页，这款插件就能满足你的需求。User Agent Switcher 直观的选项可以让你方便的设置浏览器UA，有了它，访问手机网页不再是问题。 Firefox用户点击进入User Agent Switcher下载页
