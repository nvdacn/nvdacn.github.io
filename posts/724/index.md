nvda从真正引入中国到现在也已经好多年了，使用的人也越来越多。不过，由于兼容方面的问题，一些比较老旧的操作系统即将停止维护，他们是
•Windows XP
•Windows Server 2003
•Windows Vista
•Windows Server 2008
•Windows 7 （不带SP1升级补丁）
2017.3的NVDA正式版将会是最后一个支持上述操作系统的正式版本，小伙伴可以选择升级新系统，当然也可留守老版本不升级。
